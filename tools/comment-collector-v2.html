Step 1 · 댓글 수집기 v2

유튜브 댓글 수집기

주제만 적으면 관련 영상들을 자동으로 찾아 댓글을 한 번에 모읍니다. 특정 영상 링크로도 가능합니다. 키는 어디에도 저장·전송되지 않습니다.

위 내용을 복사해서 고통지수 측정기에 붙여넣으면 끝입니다.

유튜브 API 키 발급 방법 (5분, 무료)
  1. console.cloud.google.com 접속 → 새 프로젝트 생성
  2. API 및 서비스 → 라이브러리 → "YouTube Data API v3" 검색 → 사용 설정
  3. API 및 서비스 → 사용자 인증 정보 → "사용자 인증 정보 만들기" → API 키 (API 제한사항에서 YouTube Data API v3 선택)
  4. 생성된 AIza... 키를 위 칸에 붙여넣기.

참고: 주제 검색 1회는 무료 할당량을 약 100 사용합니다(하루 10,000). 주제 수집을 하루 80번쯤 할 수 있는 양이라 개인 용도로는 충분합니다.
