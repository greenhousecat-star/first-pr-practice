# 프로젝트: 제휴 마케팅 블로그 콘텐츠 파이프라인

## 파이프라인 순서
1. topic-researcher: 글감 리서치 (Notion에서 소스 확인)
2. seo-keyword-planner: 키워드/구조 설계
3. blog-copywriter: 원고 작성
4. tone-qa-reviewer: 톤/제약조건 검수

## 톤앤매너 제약 조건
- "최고입니다", "혁신적인", "획기적인" 같은 뻔한 AI 말투 금지
- 내돈내산 후기 느낌의 솔직하고 친근한 구어체
- 제휴 링크는 [여기에 추천 링크 삽입] 형태로 본문 중간/마지막에 표시
- 소제목(###)과 글머리 기호 적극 활용

## 연결된 도구
- Notion: 리서치 자료, 소스 관리
